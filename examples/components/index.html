Form Input

A versatile form input component with validation, icons, and accessibility features. Supports different input types, error states, and custom validation. 

pfc.render('FormInput', {
  label: 'Email Address',
  type: 'email',
  placeholder: 'Enter your email',
  required: true,
  icon: 'üìß',
  helpText: 'We will never share your email',
  validation: (value) => 
    /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(value)
});

Modal Dialog

A flexible modal component with customizable headers, footers, and size options. Includes backdrop click handling and keyboard navigation. 

pfc.render('Modal', {
  isOpen: true,
  title: 'Confirm Action',
  message: 'Are you sure you want to proceed?',
  showFooter: true,
  confirmText: 'Yes, Continue',
  cancelText: 'Cancel',
  onConfirm: () => console.log('Confirmed!'),
  onCancel: () => console.log('Cancelled!')
});

Toast Notifications

Elegant toast notifications with multiple types, positions, and auto-dismiss functionality. Perfect for user feedback and status updates.

Click buttons above to show toast notifications in different positions 

pfc.render('Toast', {
  type: 'success',
  title: 'Success!',
  message: 'Your changes have been saved.',
  duration: 5000,
  dismissible: true,
  position: 'top-right',
  onDismiss: () => console.log('Toast dismissed')
});
